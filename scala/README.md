# Scala #

## Getting Started ##

### IDE (IntelliJ) ###

1. Install Java 8

   ```bash
   sudo apt update
   sudo apt install -y openjdk-8-jdk

   # Optional
   sudo update-alternatives --config java
   ```

1. Install [IntelliJ IDEA](https://www.jetbrains.com/idea/download/)

   ```bash
   sudo snap install intellij-idea-community --classic
   ```
